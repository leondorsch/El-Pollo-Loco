Class: World

World(canvas, keyboard)

Represents the game world that manages all the objects, interactions, and game logic. It handles the drawing, collisions, throwing of objects, and the status of the game.

Constructor

new World(canvas, keyboard)

Creates an instance of the game world. Initializes the character, enemies, objects, and game sounds, and starts the game loop.
Parameters:
Name Type Description
canvas HTMLCanvasElement The canvas element on which the game will be rendered.
