Class: ThrowableObject

ThrowableObject(x, y, otherDirection)

Represents a throwable object (e.g., a bottle) that the player can throw in the game. The object rotates while being thrown and plays splash animations when it hits the ground.

Constructor

new ThrowableObject(x, y, otherDirection)

Creates an instance of a throwable bottle object. Initializes the object’s image, position, and animation.
Parameters:
Name Type Description
x number The x-coordinate where the bottle is thrown from.
y number The y-coordinate where the bottle is thrown from.
otherDirection boolean Determines the direction in which the bottle is thrown.
