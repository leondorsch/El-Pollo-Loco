Class: BackgroundObject

BackgroundObject(imagePath, x)

Represents a background object in the game that moves. Extends the `MovableObject` class and provides properties and methods for managing background objects with a specific image, position, and size.

Constructor

new BackgroundObject(imagePath, x)

Creates an instance of a BackgroundObject with a specific image and x-coordinate.
Parameters:
Name Type Description
imagePath string The file path to the image to be loaded as the background.
x number The x-coordinate for positioning the background
